# ALSA — Australasian Laser Sport Association — Brand

| Token | Hex | Use |
|-------|-----|-----|
| `--brand-black` | `#0F0F0F` | Primary background |
| `--brand-black-elevated` | `#191919` | Cards, modals |
| `--brand-black-deep` | `#0A0A0A` | Footer, sidebars |
| `--brand-green` | `#00FF41` | Primary accent |
| `--brand-green-hover` | `#00CC34` | Hover states |
| `--brand-green-muted` | `#00FF4133` | Glows, subtle borders (20% opacity) |
| `--text-primary` | `#FFFFFF` | Main text |
| `--text-secondary` | `#A0A0A0` | Subtitles |
| `--text-muted` | `#666666` | Helper text |
| `--border` | `#2D2D2D` | Borders |
| `--border-subtle` | `#191919` | Subtle dividers |
