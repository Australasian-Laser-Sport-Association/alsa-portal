# Security Policy

## Reporting a Vulnerability

Please report suspected security issues privately by emailing
[committee@lasersport.org.au](mailto:committee@lasersport.org.au).

Do not open a public GitHub issue for vulnerabilities, private user data,
credentials, or suspected access-control problems.

Include:

- A short description of the issue
- Steps to reproduce, if safe to share
- The affected page, API route, or file path
- Any evidence that does not expose other users' private data

The committee will review reports and coordinate fixes before public
disclosure where appropriate.
